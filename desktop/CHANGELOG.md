# Changelog

## 0.1.0

Initial release of the Paperclip desktop app.

- Native desktop shell wrapping the Paperclip hosted platform
- System tray with show/hide and quit
- macOS overlay titlebar with native traffic light controls
- Deep linking via `paperclip://` URL scheme
- Auto-updater support
- Native notification support
- Builds for macOS (ARM64, x86_64), Windows (x86_64), Linux (x86_64)
